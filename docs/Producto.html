Class: Producto

Producto(nombre, imagen, precio, rareza, tipo, bonus)

Clase para crear todos los objetos que se pueden comprar en la tienda.

Constructor

new Producto(nombre, imagen, precio, rareza, tipo, bonus)

Crea una instancia de Producto. Si la rareza o el tipo no son validos, usa los valores 'comun' o 'consumible'.
