Module: datos/personajes.js

Este archivo contiene la definición de los personajes base que el jugador puede elegir al inicio del juego. Se exporta un array de objetos con las estadísticas iniciales y el avatar de cada personaje.

Type Definitions

DatosPersonaje

Un objeto que define las características y estadísticas iniciales de un personaje.
Type:
  • Object
Properties:
Name Type Description
id string Identificador único del personaje (ej: 'p1').
nombreClase string Nombre visible del personaje (ej: 'exploradora').
avatar string Ruta a la imagen del personaje (ej: '/imagenes/exploradora.png').
stats Object Contiene las estadísticas base del personaje.
Properties
Name Type Description
vida number Puntos de vida iniciales.
ataque number Nivel de ataque inicial.
defensa number Nivel de defensa inicial.
